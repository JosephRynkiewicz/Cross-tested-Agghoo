# Cross-tested-Agghoo
## Agghoo for DeepLearning
- The file retin_cv.py computes the aggregated holdout (Agghoo) and the test set error.
- The file retin_ct.py computes cross-tested Agghoo and the mean cross-tested set error.
- You need to have the training dataset of Rethinopathy saved in a directory "data"
- You can upload the data here: https://www.kaggle.com/datasets/josephrynkiewicz/diabetic-retinopathy-train-unzipped
- The code is designed to work with multi-GPU
